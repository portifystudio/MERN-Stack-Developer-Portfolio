# MERN Stack Developer Portfolio Skill

This skill provides a high-end, SaaS-like personal portfolio template designed for MERN stack developers.

## Features

- **Tech Stack**: Built with Vite + React + Framer Motion.
- **Design**: Deep Purple & Neon Blue theme with glassmorphism and dark/light mode.
- **Interactivity**: Custom cursor, scroll progress, animated mesh backgrounds.
- **AI Integration**: Floating AI assistant (AIChatbot.jsx) and optimized project descriptions.
- **Modularity**: All sections (Hero, About, Skills, Projects, Blog, Contact, Footer) are separate React components.

## How to Use This Template

1. **Setup**:
   - Ensure Node.js is installed.
   - Run `npm install` to install dependencies.
   - Run `npm run dev` to start the development server.

2. **Customization**:
   - **Personal Info**: Update names, roles, and socials in `src/components/Hero.jsx` and `src/components/Footer.jsx`.
   - **Projects**: Edit the `projects` array in `src/components/Projects.jsx`.
   - **Skills**: Adjust categories and proficiency levels in `src/components/Skills.jsx`.
   - **Blog**: Add or modify posts in `src/components/Blog.jsx`.
   - **AI Chatbot**: Customize the `responses` logic in `src/components/AIChatbot.jsx`.

3. **Styling**:
   - All core colors and variables are located in `src/index.css`.
   - Dark/Light mode logic is handled in `src/App.jsx`.

4. **Assets**:
   - Replace placeholder images in `src/components/` with your own professional assets.
